# WeeklyCS

## 진행 방식

1. 다음주에 발표할 4인 고르기
2. 다음주의 발표할 공통된 대주제(ex.알고리즘) 선택
3. 발표할 사람이 각자 질문을 4~5가지 정도 준비하기 (꼬리질문 포함)
4. 다음주에 모여서 각자 준비한 질문의 답변을 나누면서 설명 하기
